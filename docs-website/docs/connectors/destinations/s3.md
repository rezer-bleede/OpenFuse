# S3 Connector

The S3 destination connector writes data to Amazon S3 buckets.

## Configuration

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `bucket` | string | Yes | S3 bucket name |
| `region` | string | Yes | AWS region |
| `access_key_id` | string | Yes | AWS access key |
| `secret_access_key` | string | Yes | AWS secret key |
| `path_prefix` | string | No | Path prefix for files |

```json
{
  "destination": {
    "type": "s3",
    "config": {
      "bucket": "my-data-lake",
      "region": "us-east-1",
      "access_key_id": "${AWS_ACCESS_KEY_ID}",
      "secret_access_key": "${AWS_SECRET_ACCESS_KEY}",
      "path_prefix": "openfuse"
    }
  }
}
```

## Supported Formats

- JSON (newline-delimited)
- Parquet
